# 🤖 Asistente Analítico Avanzado - Viewnext Demo

Este proyecto es una prueba de concepto (MVP) de un agente conversacional inteligente diseñado para asistir en la automatización de procesos y el análisis de documentos empresariales.

## 🚀 Características Principales
* **Interfaz Web Interactiva:** Desarrollada íntegramente con Streamlit.
* **Motor de IA:** Integración con Google Gemini 2.5 Flash mediante *System Prompts* personalizados.
* **Ingesta de Datos (RAG):** Capacidad para procesar y analizar documentos PDF y hojas de cálculo (Excel) en tiempo real.
* **State Management:** Memoria de sesión que mantiene el contexto de la conversación fluida.

## 🛠️ Tecnologías Utilizadas
* Python 3
* Streamlit
* Google Generative AI
* PyPDF2 & Pandas
